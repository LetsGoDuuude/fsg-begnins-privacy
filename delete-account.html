Suppression de compte

Cette page vous explique comment demander la suppression de votre compte dans l'application FSG Begnins et quelles données seront supprimées ou conservées.

Comment demander la suppression

La suppression de compte se fait par e-mail à l'administrateur de l'application :

  1. Envoyez un e-mail à dev@kfmn.ch depuis l'adresse e-mail associée à votre compte FSG Begnins.
  2. Indiquez en objet : Suppression de compte FSG Begnins et précisez votre prénom et nom dans le message.
  3. Votre demande sera traitée dans un délai de 7 jours ouvrables. Vous recevrez une confirmation par e-mail une fois la suppression effectuée.
Envoyer la demande par e-mail

Données supprimées

Suite à votre demande, les données suivantes seront définitivement supprimées :

  • Votre compte d'authentification (e-mail, mot de passe)
  • Votre profil : prénom, nom, rôle, jours d'entraînement
  • Votre token de notification push
  • Vos absences déclarées

Données conservées

Certaines données peuvent être conservées à des fins d'archivage interne de la section :
  • Les remarques sur les gymnastes que vous avez rédigées — l'auteur sera anonymisé (mention « Moniteur »).
  • Les fiches d'entraînement auxquelles vous avez contribué — conservées dans l'historique de la section.
  • Les notes de concours saisies par vos soins — conservées dans les résultats archivés.
Toutes les données personnelles permettant de vous identifier (nom, e-mail, compte) sont supprimées dans les 7 jours suivant votre demande. Les données pédagogiques liées aux gymnastes sont anonymisées et conservées.
